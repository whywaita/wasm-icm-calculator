# 使い方ガイド

## ブレイクイーブン分析

ブレイクイーブン分析パネル（デフォルトでは折りたたみ）を使用して、ICMエクイティに基づいてトーナメントエントリーが利益的かどうかを評価できる。

### 入力項目

- **エントリーフィー**: トーナメント参加に支払う合計金額（例: $110）。
- **レーキ %**: エントリーフィーのうち運営が徴収する割合（デフォルト: 10%）。バイインとレーキはこの2つの値から自動計算される。
- **スターティングチップ**: トーナメント開始時に各プレイヤーが受け取るチップ数（例: 10,000）。

### 出力（結果テーブル）

- **エントリーフィー**: 参照用のエントリーフィー値。
- **損益 (P/L)**: `ICM$ - エントリーフィー`。正の値は利益的なポジションを示す。
- **ICMプレミアム**: ICMエクイティとチップEVの比率（`ICM$ / (スタック × バイイン / スターティングチップ)`）。1.0より大きい場合、そのプレイヤーのチップはリニアなチップ価値より高い。

### チャート連携

ブレイクイーブン有効時:

- **エクイティ棒グラフ**: 赤い破線の水平線がエントリーフィーレベルを示す。バーは色分けされる: ブレイクイーブン以上のプレイヤーはteal、以下はred。
- **ICMプレッシャーカーブ**: 赤い破線の水平線がエントリーフィーレベルを示し、カーブがブレイクイーブンと交差する箇所が可視化される。

## エクイティ棒グラフ

各プレイヤーのICMエクイティを縦棒グラフで表示する。

- **スタンダードモード**: ICM$を示すtealの単色バー。
- **バウンティ/PKOモード**: ICM$（teal）とバウンティエクイティ（amber）の積み上げバー。
- **ブレイクイーブン有効時**: プレイヤーのエクイティがエントリーフィーを超えるかどうかでバーの色が変化する。凡例が表示される。

## ICMプレッシャーカーブ

チップスタックサイズとICM$値の関係を可視化し、チップの限界価値逓減（ICMプレッシャー）を示す。

- **X軸**: チップ数（千区切り表示）。
- **Y軸**: ICM$値（ドル表記）。
- カーブはチップを追加するほどICM$の増加が小さくなることを示す — ICMの基本的な洞察。
- **ブレイクイーブン有効時**: エントリーフィーレベルに赤い破線が表示され、カーブ上のブレイクイーブン閾値を示す。
